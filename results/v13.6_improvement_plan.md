# v13.6问题诊断与改进方案

## 🔍 核心问题确认

### 问题1：候选龙头进入池子，但没有买入
**现象**：
- 长飞光纤(601869.SH)多次进入龙头候选池
- 但没有触发买入信号

**原因**：
- 现有买点逻辑过于保守：
  - `ice_point`（冰点）：要求涨停分位<20%，太罕见
  - `technical_ma5`（回踩5日线）：需要股票回调，但龙头可能一直沿5日线上涨
  - `weak_as_strong_courage`（弱为强）：需要前一天涨停，条件苛刻

- **缺失核心买点**：板块起爆分歧

### 问题2：买入的都是非龙头股票
买入次数Top 15中，没有一只是真龙头（中际旭创、新易盛、长飞光纤等）

## 💡 改进方案（基于用户指导）

### 1. 增加"板块起爆分歧"买点 ⭐核心

**逻辑**：
```
1. 板块识别成功（聚类出热点板块）
2. 板块内股票开始涨停潮（首次冲高）
3. 次日分歧回调（部分龙头小幅回调，部分后排跟涨）
4. 买入龙头（在分歧点低吸龙头）
```

**实现要点**：
- 监控板块内涨停股票数
- 识别"首次涨停潮"（历史窗口内第一次出现>=3只涨停）
- 次日分歧：龙头回调2-5%，但不破5日线

### 2. 增加"牛市直接追涨"买点

**判断牛市**：
- 指数连续上涨（如5天内4天上涨）
- 大盘量价齐升（成交量放大）

**买点**：
- 龙头突破新高
- 不等回调，直接买入

### 3. 优化龙头评分权重

**当前**：累计涨幅50分 + 提前启动30分 + 极限量20分

**调整后**：
- 累计涨幅：40分（板块内最高）
- 提前启动：40分（在板块前1/3时间段的表现）
- 极限量：20分（与自己历史对比）

### 4. 优化卖出逻辑

**增加"板块高潮"卖出信号**：
```
- 板块后排股票集体涨停（>=70%涨停）
- 龙头当日涨幅<5%（赚钱效应转移）
- → 卖出龙头
```

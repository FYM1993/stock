## 🔍 可能的深层问题

### 问题1：板块识别问题
**猜测**：新易盛可能根本没进入"热点板块"

**原因**：
```python
# 板块识别条件（line 447-456）
1. 相关性>0.7：同板块股票走势相似
2. 平均涨幅>阈值：8天需要>5%
3. 至少3只股票
```

**可能的情况**：
- 新易盛是**龙头独角兽**，板块内其他股票涨幅不够
- 或者板块内股票数<3只
- 导致整个板块没被识别出来
- **新易盛根本没进入候选池评分环节！**

### 问题2：评分逻辑问题
**当前评分（line 528-576）**：

  - 量能更大（+10分）
  - 或提前启动更早（+5分）
  - 总分40-45分 > 新易盛30分

## 💡 修复方案

### 方案1：增加全市场区间涨幅Top10（推荐）

```python
def _update_leader_candidates(self, date: str):
    # Step 1: 先获取全市场区间涨幅Top10
    market_top10 = self._get_market_top_gainers(date, top_n=20)
    # 扩大到20只，保证不遗漏
    
    # Step 2: 识别热点板块
    sectors = self._identify_hot_sectors(date)
    
    # Step 3: 合并候选
    candidates = set()
    
    # 从热点板块中选
    for sector_stocks in sectors.values():
        candidates.update(sector_stocks)
    
    # 强制加入全市场Top10（防止遗漏）
    candidates.update(market_top10)
    
    # Step 4: 在候选集中评分排序
    scored = self._calculate_leader_scores(candidates, date)
    self.leader_candidates = sorted(scored, key=lambda x: x['score'], reverse=True)[:10]
```

**效果**：
- 新易盛即使板块没被识别，也会因为区间涨幅高进入候选
- 符合《龙头低吸.md》的原始思想

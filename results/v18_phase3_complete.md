### 2. 重写买入信号检查

**文件**：`strategies/leader_low_absorption.py`

**方法**：`_check_all_buy_signals()` (重写约180行)

**v18逻辑**：

```python
def _check_all_buy_signals(symbol, date, market_regime):
    signals = []
    
    # 弱势市场：不参与
    if market_regime == WEAK:
        return signals
    
    # 强势市场：追涨为主
    if market_regime == STRONG:
        100分: 追涨龙头 - 涨幅>5% + 在MA5上方
         90分: 板块起爆 - 涨幅>5% + 放量1.5倍 + 突破MA5
         80分: 强势突破 - 突破10日前高
    
    # 震荡市场：低吸为主
    elif market_regime == OSCILLATE:
        100分: 情绪冰点低吸 - 市场最低迷 + 个股下跌
         90分: 龙头回踩 - 回调到MA5附近
         80分: 板块起爆 - 放量突破
         70分: 缩量回调 - 缩量回踩MA5
    
    # 按优先级排序
    return sorted(signals, by priority)
```

### 3. 新增辅助方法

#### 3.1 板块起爆分歧检查

```python
def _check_sector_breakout(symbol, date, stock_data):
    """
    检查板块起爆分歧
    
    标准：
    1. 当日涨幅>5%
    2. 成交量>5日均量的1.5倍
    3. 股价在MA5上方
    """
```

**用途**：
- 捕捉板块首次启动的机会
- 强势和震荡市场都适用
- 强势优先级90分，震荡优先级80分

#### 3.2 回踩买入信号检查

```python
def _check_pullback_signal(symbol, stock_data):
    """
    检查回踩买入信号
    
    标准：
    1. 股价回调到MA5附近（0.95-1.02倍）
    2. 当日涨幅在[-2%, +3%]
    3. 前期有过上涨（5日内最高价>当前价5%）
    """
```

**用途**：
- 震荡市场的主要买点
- 在龙头回调时低吸
- 优先级90分（仅次于冰点）

### 4. 买入信号优先级设计

#### 4.1 强势市场（追涨为主）

| 信号 | 优先级 | 触发条件 | 说明 |
|------|--------|---------|------|
| 追涨龙头 | 100 | 涨幅>5% + 在MA5上方 | 直接追最强势的龙头 |
| 板块起爆 | 90 | 涨幅>5% + 放量1.5倍 | 板块首次启动 |
| 强势突破 | 80 | 突破10日前高2% | 突破关键阻力位 |

**逻辑**：
- 强势市场不怕追高
- 优先买入最强势的股票
- 100%仓位集中持有

#### 4.2 震荡市场（低吸为主）

| 信号 | 优先级 | 触发条件 | 说明 |
|------|--------|---------|------|
| 情绪冰点 | 100 | 市场冰点 + 个股下跌 | 市场最恐慌时买入 |
| 龙头回踩 | 90 | 回调到MA5附近 | 强势股回调买入 |
| 板块起爆 | 80 | 涨幅>5% + 放量 | 捕捉板块启动 |
| 缩量回调 | 70 | 缩量回踩MA5 | 温和回调买入 |

**逻辑**：
- 震荡市场注重低吸
- 优先在市场恐慌时买入
- 40%仓位分散风险（2只）

## 📊 对比分析

### v17问题

**买入信号混乱**：
```
强势市场：追涨、冰点、板块起爆、突破...（优先级不清）
震荡-板块效应：板块起爆、冰点、回踩...（优先级不清）
震荡-板块轮动：前龙修复、缩量回踩...（优先级不清）
→ 结果：买入决策不明确，容易买错
```

### v18改进

**买入信号清晰**：
```
强势市场：追涨(100) > 板块起爆(90) > 突破(80)
震荡市场：冰点(100) > 回踩(90) > 板块起爆(80) > 缩量(70)
→ 结果：买入决策明确，优先级清晰
```

### 优先级意义

| 分数段 | 含义 | 说明 |
|--------|------|------|
| 100分 | 最优买点 | 最符合当前市场环境的买点 |
| 90分 | 优质买点 | 次优选择，质量也很高 |
| 80分 | 良好买点 | 可以考虑，但优先级较低 |
| 70分 | 一般买点 | 有机会但不是最佳时机 |

**作用**：
1. 当有多只股票同时触发信号时，按优先级排序
2. 优先买入优先级最高的股票
3. 避免盲目买入低优先级的股票

## 🔍 实例说明

### 场景1：强势市场，5只股票同时触发信号

```
股票A: 追涨龙头(100分) - 涨幅7%，龙头评分第1
股票B: 板块起爆(90分) - 涨幅8%，放量2倍
股票C: 强势突破(80分) - 突破前高3%
股票D: 追涨龙头(100分) - 涨幅6%，龙头评分第2
股票E: 板块起爆(90分) - 涨幅5%，放量1.6倍

排序：A(100) > D(100) > B(90) > E(90) > C(80)

买入决策：
- 强势市场只买1只，100%仓位
- 选择优先级最高且龙头评分最高的：股票A
```

### 场景2：震荡市场，6只股票同时触发信号

```
股票A: 情绪冰点(100分) - 冰点+下跌2%
股票B: 龙头回踩(90分) - 回调到MA5，前期涨幅高
股票C: 板块起爆(80分) - 涨幅6%，放量
股票D: 情绪冰点(100分) - 冰点+下跌1%
股票E: 缩量回调(70分) - 缩量回踩
股票F: 龙头回踩(90分) - 回调到MA5

排序：A(100) > D(100) > B(90) > F(90) > C(80) > E(70)

买入决策：
- 震荡市场买2只，各40%仓位
- 选择优先级最高的：股票A + 股票D（都是冰点低吸）
```

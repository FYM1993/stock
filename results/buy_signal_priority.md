# 买点优先级和逻辑说明（v14）

## 📊 买点检测顺序（优先级从高到低）

### 1. 情绪冰点低吸 (ice_point)
**条件**：
- 涨停数处于近60天的20%分位以下
- 龙头当日跌幅<3%

**触发逻辑**：市场极度弱势，恐慌性下跌后的反弹机会

---

### 2. 板块起爆分歧 ⭐新增 (sector_outbreak_divergence)
**条件**：
- 股票在热点板块中
- **前一日板块内>=3只涨停**（板块起爆）
- **今日龙头回调2-5%**（分歧）
- **不破5日线**（强势回调，收盘价>=5日线*0.97）

**触发逻辑**：
```
板块首次涨停潮 → 次日龙头分歧回调 → 低吸龙头

这是板块启动后的黄金买点，既确认了板块热度，
又在分歧时以相对低位买入龙头
```

**重要性**：这是文档《龙头低吸.md》强调的核心买点！

---

### 3. 板块分歧低吸 (sector_divergence)
**条件**：
- 昨日涨停数在高位（>60%分位）
- 今日涨停数降幅>15%
- 龙头小幅回调或横盘（-6% ~ +3%）

**触发逻辑**：市场整体分歧，龙头回调

---

### 4. 弱为强之胆 (weak_as_strong_courage)
**条件**：
- 市场强势（涨停数>50%分位）
- 龙头回调幅度-5% ~ 1%
- 价格在5日线附近（0.95~1.1倍）

**触发逻辑**：市场强势时，板块小弟涨停，龙头回调买点

---

### 5. 技术面低吸 (technical_ma5)
**条件**：
- 缩量回踩5日线（0.95~1.05倍）
- 成交量缩小<0.9倍前5日平均

**触发逻辑**：技术面健康回调

---

### 6. 强势突破 (strong_breakthrough)
**条件**：
- 市场强势（strong模式）
- 股价突破10日新高
- 涨幅2%-9%

**触发逻辑**：牛市追涨龙头

---

## 🔍 执行逻辑

### 串行检测（Early Return）
代码采用**串行检测，满足任一条件即返回**：

```python
if 冰点:
    return True, 'ice_point'

if 板块起爆分歧:  # ⭐新增
    return True, 'sector_outbreak_divergence'

if 板块分歧:
    return True, 'sector_divergence'
    
if 弱为强之胆:
    return True, 'weak_as_strong_courage'
    
if 技术面低吸:
    return True, 'technical_ma5'
    
if 强势突破:
    return True, 'strong_breakthrough'

return False, ''
```

**优点**：
- 简单高效
- 优先级明确（越靠前优先级越高）
- 避免多个信号冲突

**缺点**：
- 龙头分数低的候选股，可能永远轮不到检测
- 满仓后，后续候选股直接跳过

---

## 💡 为什么长飞没买入？

**根本原因**：遍历问题，不是买点问题

1. **候选池排序**：按龙头分数排序，长飞可能不是Top1-3
2. **遍历中断**：前面的股票买入后，达到最大持仓数（3只），停止遍历
3. **结果**：长飞虽然在候选池（有分数），且可能满足买点，但轮不到检测

**举例**：
```
2026-02-06:
候选池: [002491.SZ(85分), 600916.SH(85分), ..., 601869.SH(第7名)]
         ↓
遍历到002491.SZ → 触发technical_ma5 → 买入
         ↓
已有2只持仓 + 本次1只 = 3只
         ↓
达到max_positions，停止遍历 ❌
         ↓
长飞没机会检测
```

## 🎯 解决方案

### 短期（已实现）
